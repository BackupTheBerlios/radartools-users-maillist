Starting with this version, RAT is distributed under a 'real' open-source 
license, more precisely under the Mozilla public license version 1.1 (MPL 
1.1). 

Of course there is also some 'fresh' functionality in this release:
- New routines: PolDInSAR phase and height estimation from MB-PolInSAR data
- New routine : Individual basis transformation per pixel 
- New routine : Branch-cut (Goldstein) phase unwrapping
- New routines: Import of ASF SLC formats (ERS, JERS, RADARSAT-1)
- Improved routines: Corrected RoA edge detectors
- Improved routines: Possibility to save optimal scattering mechanism vectors 
- Various small improvements and bug fixes

Cheers,

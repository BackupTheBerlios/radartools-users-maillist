Hmm, came up with a simple work around.  But it seems that  that my 
header data is different than what is expected in open_asf_slcs.pro.

 On line 81, there is this:

    if header[48] EQ 69 AND header[49] EQ 50  THEN spacecraft = 'ERS-2'

but my header values are header[48] = 69 and header[49] = 82 - note sur 
what the discrepancy there is.

Anyway, my work around was to simply initialize the 'spacecraft' string 
variable:

spacecraft = 'UNK' prior to line 78 and it runs fine (the 'spacecraft' 
string seems to be just information for the title bar of the dialog window.)

Dear All,

In fact I was not aware that RAT is not running on 64-bit Windows. I know, 
however, that there are some issues with newer releases of IDL. Since long 
time we had the plan to complete another release including such fixes, but 
unluckily it never happend. The problem is that current development is mostly 
stalled since all core developpers are busy in their jobs. But I am willing to 
reserve some time to complete this release!

Therefore: Can you descripe what exactly is the problem in 64bit Windows? 
Since RAT does fine on 64bit Linux it is probably quite easy to fix.

Secondly, there are some ideas floating around to initiate a 'new' RAT project 
in python. Some code is already existing, but time is lacking...

Regards,
Andreas

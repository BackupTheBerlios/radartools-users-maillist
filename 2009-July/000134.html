Development has been a bit slow in the last time, since most developers have 
been very busy with other activities. However, RAT is not dead and continues 
to develop! Here comes the newest release:

- New routine: Import of PALSAR ERSDAC (CEOS and VEXCEL) format
- New routine: Range antenna profile removal
- New routine: CDA sidelobe suppression / resolution enhancement
- New routine: Polarimetric SERD/DERD parameters
- New routine: Ainsworth PolSAR calibration
- New routine: Polarimetric SNR estimation
- Improvemed batch mode
- Faster preview calculation & zoom
- Inline documentation based on IDLdoc
- Various small improvements and bug fixes

Download the code at: http://radartools.berlios.de/download.php

RAT has found a new place for its website, you can choose between

http://radartools.berlios.de
http://rat.nought.de

Have fun,
Andreas

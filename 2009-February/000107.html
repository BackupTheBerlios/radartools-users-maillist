Hi, 

1) I think the problem is just that RAT uses a clever preview calculation... 
In fact, I think  the image probably _is_ speckled as you see it in TIFF. The 
RAT window presums the full resolution image for display which acts like a 
speckle filter. Try to use the zoom tool in RAT, the you see the "real" 
image.
2) You need some calibration values (from the COSAR header, I don´t know). You 
won´t be able to do the conversion with RAT.
3) You again need some values (from the COSAR header, I don´t know). However, 
there is a simple slant-to-ground conversion in RAT for flat terrain and 
neglecting Earth´s curvature. Maybe this is enough for your purpose?

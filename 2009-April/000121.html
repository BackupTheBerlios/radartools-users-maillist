Dear Qingsong Wang,

you can find the source code of rat e.g. at
http://srv-43-200.bv.tu-berlin.de/rat/download.php
or
http://svn.berlios.de/wsvn/radartools/trunk/
If you want to find out, in which file a certain compiled function is
located, then you can use the following idl command (e.g. for function
cc_opt):
print, routine_info("cc_opt",/source,/function)
if "cc_opt" is a function. If it's a procedure, you have to use
print, routine_info("cc_opt",/source)
In this case, most of functions you ask for are located in the file
polinsar/polin_lib.pro:
http://svn.berlios.de/wsvn/radartools/trunk/polinsar/polin_lib.pro

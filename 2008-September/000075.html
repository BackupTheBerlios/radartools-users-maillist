Hi Li Ting Wei,

it's nice that you use RAT and want to extend it.

> Firstly.Recently,I have written a IDL programm which is used to estimate
> the coherence and I want to add to the RAT,but I do not know how to do.Could
> I add my IDL progamm to the RAT? If could,please tell me how to?
>
At the page:
http://www.cv.tu-berlin.de/rat/manual.php
there are several tutorial/info pages about how to extend Rat. First you
should consult
http://www.cv.tu-berlin.de/rat/manual.php?rat_inputfilter
You need of course good IDL programming skills, but this you probably have.
You can also check other functions, how they are written. You can just take
one function which does something similar to yours and try to modify it so
that you can include your own function. If you are writing a coherence
estimation function for InSAR, you could start e.g. with
http://svn.berlios.de/wsvn/radartools/trunk/insar/coh_mean.pro
or, if it's more for PolInSAR you can check
http://svn.berlios.de/wsvn/radartools/trunk/polinsar/polin_coh_mean.pro
If it all works and your program brings interesting new functionality we can
help you to include it into the next RAT release.

Secondly.I want to use the two complex SAR iamges data which is registrated
> by the RAT,How can I get them? How the two registrated SAR images are saved?
>
If you mean what's the file format of RAT data files, you can check it also
here (see the links to "RAT file format"):
http://www.cv.tu-berlin.de/rat/manual.php

Good luck!
Best regards,
Maxim

Hi Wei,

1 - To add a new function to RAT menu you need to edit the rat.pro file and simply add your line like the example bellow in the CASE logic:

'Coherence estimation.Boxcar': polin_coh_mean	

The first string is the menu text and the second string is the function name.

The function file has to be compiled already or to be in the search path.

2 - All data processed by RAT is in its special structure. It is not a simple one but you can learn how it is implemented by following the code and inspecting variables while running a test case. That's how I'm learning it right now. How easy it will be depends on your expertise with IDL.

3 - To be more helpfull and brief, I need more details from you.


Good luck!

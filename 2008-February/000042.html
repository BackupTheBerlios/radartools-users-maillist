Hi,

RAT is a software for PolSAR and InSAR processing. While the PolSAR part is 
well developed, the InSAR part isn't. You can do many InSAR processing tasks, 
but specific tools for handling the orbit geometry, the geocoding, etc. are 
missing up to now.

You're right that ESA data is usally in Doppler-Zero geometry. PALSAR data 
might be not, but this is not a big problem - co-registration should fix 
this.

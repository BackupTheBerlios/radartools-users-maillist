>> I'd like to make polarimetric interferometry images of ALOS/PALSAR.
>> Let me know how to make them using RAT.

Please have a look at the Rat homepage (->Manual). E.g. the 2005/2007  publications concerning PolInSAR.
Note that Rat doesn’t provide special routines for ALOS/PALSAR PolInSAR data processing (like registration/coregistration, etc.), but has only some generic functionality, which can be used for further data analysis.

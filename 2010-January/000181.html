> On the Linux command line, type "idl -vm", this will start up the virtual
> machine. In the file selector, select the "rat.sav" file provided in the
> binary version of RAT. Or type "idl -vm=rat.sav" directly (as written on the
> RAT website!) Just to avoid problem, maybe delete before a possible directory
> called ".rat" in your home directory. Hope that helps!
